# ✨v0.5.7 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。ZZ


## 🚀 新功能

- **通用智能体：**
  - 新增数字员工

- **Skill：**
  - 支持回退草稿中间版本

- **本体智能体：**
  - 数据连接支持PostgreSQL数据库类型。
  - 数字员工上线【数据采集员】支持从图片或文档（PDF、Word、Excel）中提取结构化数据替代人工自动录入数据库

- **智能体：**
  - 添加工具显示描述

- **Skill广场：**
  - 支持查看skill下载量和发送量

- **资源库：**
  - 内置工具新增【文件解压】工具

- **工作流：**
  - Http节点支持接收流式返回
  - 工作流、对话流创建调整为统一入口
  - 文档生成节点增加md和html格式，现有的docx、pdf、txt可输出md渲染后的结果


## 📈增强功能

- **用户体验：**
  - 优化多人同时登录，登录页验证码错误
  - 知识库搜索后返回，携带搜索历史
  - 平台配置页面卡顿问题

- **工作流：**
  - 工作流节点增大超时配置


## 🛠️ 错误修复

- 文本类知识库可上传多模态文件
- API调用总次数和api调用详情的条数不一致
