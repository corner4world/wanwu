# ✨v0.5.10 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。


## 🚀 新功能

- **本体智能体：**
  - 数据连接支持Oracle数据库类型

- **智能体：**
  - 支持需要配置变量后使用的Skill。安全风险提示：如果智能体被不当使用，您为skill配置的变量有泄露风险，请慎重评估变量的敏感级别并选择适当的智能体发布范围
  - 支持多文件上传
  - 新增内置工具文档解析chatdoc，默认选中。如您有其他类文档解析工具，可取消勾选chatdoc后使用

- **模型管理：**
  - 联通元景升级OCR类型，取消文档解析类模型。提示：如您之前使用了文档解析模型，请在联通元景-OCR类型模型中重新导入，方可继续使用

- **Skill：**
  - 支持手动“安全审核”，我们强烈建议您，在发布Skill前，进行安全审核

- **渠道管理：**
  - 支持配置微信、钉钉渠道，关联平台内智能体使用


## 📈增强功能

- **登录加密改造**
  - 对称加密改为非对称加密

- **知识库：**
  - 文本知识库文档解析方式调整为文字解析和OCR解析两类

- **插件工具：**
  - 新增内置工具：文档解析

- **Skill：**
  - 新增多个内置Skill


## 🛠️ 错误修复

- **本体智能体：**
  - 前端页面导航栏收缩后logo与平台配置不同步问题

- **模型管理：**
  - 模型体验会话标题过长
  - 模型体验进入会话界面后，不能跳转至最新记录
