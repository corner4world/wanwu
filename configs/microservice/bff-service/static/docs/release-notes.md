# ✨v0.6.0 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。


## 🚀 新功能

- **模型管理：**
    - OpenAI-API-compatible供应商支持短语音识别类模型导入

- **管理员中心：**
    - 原“设置”，升级为管理员中心。仅管理员有权限使用。全面升级组织、用户、角色管理界面。
    - 系统管理员可创建全局角色
    - Oauth管理划转至管理员中心
    - 角色管理新增支持控制“开源仓库”可见性

- **Skill：**
    - skills创建时，每修改一次skill，历史skill自动删除

- **工作流：**
  - 支持需要配置变量使用的Skill

- **渠道管理：**
  - 支持关联平台内通用智能体、数字员工使用
  - 已关联的渠道进行编辑，支持重新扫码绑定

- **本体智能体：**
  - 数字员工会话页嵌入配置管理，可一键复制嵌入js脚本到业务方应用中嵌入数字员工对话页
  - 数据连接支持SQLServer数据库类型


## 📈增强功能

- **通用智能体：**
  - 对话列表改为最新使用对话置顶

- **应用广场：**
  - 对话列表改为最新使用对话置顶
