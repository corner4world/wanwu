# 批量新增文档切片API

知识库批量新增文档切片支持通过上传csv文件的方式，对处理完成的文档批量新增分片内容。批量导入为异步执行，可通过获取文档分片列表API查看导入进度与结果。

csv文件格式为两列：第一列为分段内容，第二列为关键词（多个关键词用英文逗号分隔并用英文双引号包裹），示例：

```csv
分段内容,关键词
2024 年新款智能手表支持 24 小时心率监测、睡眠质量分析，内置蓝牙通话功能，续航可达 7 天,"智能手表,心率监测"
```

## 请求接口

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/batch/create` |
| 调用方式 | 同步调用（导入任务异步执行）                                    |
| 字符编码 | UTF-8                                                        |
| 请求类型 | POST                                                         |
| 鉴权方式 | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json`                             |
| 响应格式 | `Content-Type: application/json`                             |

## 请求参数

| Parameter    | Required  | Type   | Instructions                                                 |
|---------------|----| ------ |--------------------------------------------------------------|
| docId | 是  | string | 文档Id，仅支持处理完成状态的文档 |
| fileUploadId | 是  | string | csv文件上传Id，通过上传文档API（`/openapi/v1/file/upload/direct`）上传csv文件后返回的fileId |

## 响应参数

| Parameter          | Type         | Instructions   |
|--------------------| ------------ |--------------|
| code               | int          | 状态码        |
| data               | object       | 返回数据   |
| msg                | string       | 返回信息       |

## 请求示例

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/batch/create' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}' \
--data '{
    "docId": "2061726711199830016",
    "fileUploadId": "59bfaef5-8609-48ad-a69d-305fcd221e93.csv"
}'
```

## 响应示例

```json
{
  "code": 0,
  "data": null,
  "msg": ""
}
```

## 常见错误码

| Code   | Instructions                                                 |
|--------|--------------------------------------------------------------|
| 142010 | 上传文档必须是csv，请修改后重试 |
| 145010 | 新增文档分段失败，请稍后重试（如文档非处理完成状态） |
