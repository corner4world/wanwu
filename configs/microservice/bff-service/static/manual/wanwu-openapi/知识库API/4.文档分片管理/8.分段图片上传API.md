# 分段图片上传API

## 知识库分段图片上传API
知识库分段图片上传支持上传分段图片并返回markdown格式的图片地址，返回的地址可直接拼接进分片内容中（配合新增/更新文档切片API使用）。仅多模态知识库可使用该接口，支持png/jpg/jpeg格式的图片。

### 请求接口

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/image/upload` |
| 调用方式 | 同步调用                                                      |
| 字符编码 | UTF-8                                                        |
| 请求类型 | POST                                                         |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}`                 |
| 请求格式 | `Content-Type: multipart/form-data;boundary=XX`              |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

| Parameter    | Required  | Type   | Instructions                     |
|--------------|---------- | ------ |------------------------|
| knowledgeId  | 是        | Text   | 知识库Id，仅支持多模态知识库 |
| files        | 是        | File   | 图片文件，支持png/jpg/jpeg格式 |

### 响应参数

| Parameter  | Type     | Instructions   |
| ---- | ------ | ---- |
| code | int    | 状态码  |
| data | object | 上传图片信息 |
| msg  | string | 返回信息 |

#### fileList中详细字段

| Parameter | Type   | Instructions   |
| --------- | ------ | ---- |
| fileUrl   | string | markdown格式的图片地址，如`![图片名称](图片下载地址)`，可直接拼接进分片内容 |

### 调用示例

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/image/upload' \
--header 'Authorization: Bearer {API Key}' \
--form 'knowledgeId="2061726506710468608"' \
--form 'files=@/home/example.png'
```

### 响应示例

```json
{
    "code": 0,
    "data": {
        "fileList": [
            {
                "fileUrl": "![example.png](http://localhost:8081/minio/download/api/rag-doc/ac0ecdf1-6958-4e95-bdb2-e6a21912f7fa.png)"
            }
        ]
    },
    "msg": ""
}
```

### 常见错误码

| Code   | Instructions                                                 |
|--------|--------------------------------------------------------------|
| 145022 | 当前知识库不支持插入图片，仅多模态知识库可插入图片 |
| 145023 | 仅支持上传png/jpg/jpeg格式的图片 |
