# 新增文档切片API

## 知识库新增文档切片API
知识库新增文档切片支持对处理完成的文档单条新增分片内容，可同时设置分片关键词。多模态知识库的分片内容中可插入图片markdown（图片地址通过分段图片上传API获取），普通知识库分片内容不允许包含图片。

### 请求接口

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/create` |
| 调用方式 | 同步调用                                                      |
| 字符编码 | UTF-8                                                        |
| 请求类型 | POST                                                         |
| 鉴权方式 | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json`                             |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

| Parameter    | Required  | Type   | Instructions                                                 |
|---------------|----| ------ |--------------------------------------------------------------|
| docId | 是  | string | 文档Id，仅支持处理完成状态的文档 |
| content | 是  | string | 分段内容，不能为空且长度不能超过文档配置的最大分段长度；多模态知识库可包含图片markdown（如`![图片名称](图片地址)`），普通知识库不允许 |
| labels | 是  | string array | 关键词列表，无关键词时传空数组`[]` |

### 响应参数

| Parameter          | Type         | Instructions   |
|--------------------| ------------ |--------------|
| code               | int          | 状态码        |
| data               | object       | 返回数据   |
| msg                | string       | 返回信息       |

### 请求示例

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/create' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}' \
--data '{
    "docId": "2061726711199830016",
    "content": "这是一段新增的分段内容",
    "labels": ["关键词1", "关键词2"]
}'
```

### 响应示例

```json
{
  "code": 0,
  "data": null,
  "msg": ""
}
```

### 常见错误码

| Code   | Instructions                                                 |
|--------|--------------------------------------------------------------|
| 145010 | 新增文档分段失败，请稍后重试（如文档非处理完成状态） |
| 145011 | 分段长度超过最大值 |
| 145020 | 文档分段内容为空 |
| 145022 | 当前知识库不支持插入图片，仅多模态知识库可插入图片 |
