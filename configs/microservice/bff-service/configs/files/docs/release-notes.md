# ✨v0.6.0 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。


## 🚀 新功能

- **模型管理：**
    - 百度千帆供应商新增导入OCR类型（Paddle OCR）
    - 联通元景供应商OCR类型模型升级，支持本地化部署MinerU3.0模型接入（已开源），万悟OCR服务部署方式文档如下：https://github.com/UnicomAI/DocParserServer

- **对话列表：**
    - 支持搜索历史对话

- **OpenAPI：**
    - 新增知识库单条新增 / 批量新增(csv) / 更新 / 删除 / 分段图片上传接口

- **本体智能体：**
  - 数字员工支持定时推送任务设置（支持定时调度数字员工并将执行结果推送钉钉/微信）
  - 数据连接支持Excel文件导入功能（支持在PostgreSQL类型数据库中导入Excel）


## 📈增强功能

- **智能体：**
  - 文件上传样式优化

- **导航栏：**
  - 支持展示组织Logo


## 🛠️ 错误修复

- 通用智能体连续对话时模型报错
- 工作流大模型节点增加思考过程
- 工作流变量聚合节点删除Group时展示错乱
- 工作流变量聚合节点中String类型输入空字符显示问题
